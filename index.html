Renole Engelang

A language you speak with arrows.

Renole is an experimental Engelang that is spelled out with only four symbols*:

  • <
  • >
  • ^
  • v
*note that the symbols we use are the simplest way to type arrows, you may use anything that is an arrow - like unicode arrows.

Sounds easy? Well that's only the letters. Different combinations make different sounds, nouns and adjectives, etc... This collection of web pages aims to explain how the language works.
